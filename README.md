# 使用方法

## テーマをダウンロード

テーマのフォルダ名は「oct-bootstrap-theme-col1」にして下さい。

## 子テーマをダウンロード

[GitHub](https://github.com/feelings-for-peperoncino/oct-bootstrap-theme-col1-child)

## WordPressにアップロード

親テーマと小テーマをWordPressにアップロードして下さい。

## 小テーマを有効化

親テーマはアップデートが行われる場合があります。

子テーマを有効化しご利用下さい。
